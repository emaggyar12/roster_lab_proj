# Full Roster Lab Project

* `models_dir/` contains models used for BPR prediction, only outputs that were useful are included.
* `roster_lab/` contains site construction.
* `data_pulls/` and `scrapers_web/` contain scrapers/API requests for data used.
* `roster_lab/data/data_sources.txt` contains glossary of sources of data and how they were used on the site.
* Email aharsha@vols.utk.edu for the .db files used to source the majority of the data
